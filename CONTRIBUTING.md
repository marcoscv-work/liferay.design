# Contributing

Most of the site is content under [`src/markdown/`](./src/markdown) — editing an
article, event, team member or Lexicon page is just editing markdown.

1. Run the site locally (see the [README](./README.md)): `cd astro && npm install && npm run dev`.
2. Edit the relevant markdown in `src/markdown/` (or a page/component in `astro/src/`).
3. Open a pull request — Netlify posts a deploy preview automatically.

Questions or something broken? Open an [issue](https://github.com/liferay-design/liferay.design/issues).
